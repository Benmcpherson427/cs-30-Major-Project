# **Major Project Propoal

## Project Description
A Pokemon battle simulator

## Needs to Have List

- Moves for each pokemon

- A start screen

- HP, ATK, DEF, SPD, SPA, SPE stats
- Text boxes saying what moved was used, if the move missed, if the pokemon fainted
